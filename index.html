ABOUT

My studies honed my creativity and gave me a skillset to visually communicate using typography, composition, and color. In my growth as a designer I understood I had to expand my skills since print is consumed on the web more.

I enrolled Macomb Community College Inetractive Web Media program where I was introduced to Html, Css, flash, Javascript, and different frameworks. MCC's cirriculum gave me general understanding of the web which narrowed the path I should focus on. I created several sites using HTML, CSS, and some jquery plugins. I wanted I a deeper understanding of web technologies and began to seriously pursue a career in web development so I enrolled in Winter 2017 Grand Circus Front End Developer Bootcamp. The bootcamp ushered me into the Json framework and delve futher into bootstrap. Collaborative work was a focus in result I contributed to 2 different applications using version control, Angular, and Jquery.

Problem solving is a universal skill that enhances your execution and expertise in any field of focus. I have learn this in my work experience freelance graphic design and now web development.My goal is to work for or with a company that will give me the opportunity to strengthen my skill as a web developer.
